# 📊 Student Marks Analyzer

## About
A Data Science project that analyzes student marks
and creates visualizations using Python.

## Tools Used
- Python 🐍
- Pandas 📊
- Matplotlib 📈
- Seaborn 🎨
- NumPy 🔢

## Features
- ✅ Analyzes marks of 10 students
- ✅ Shows top and lowest performing students
- ✅ Creates Bar Chart, Heatmap & Pie Chart
